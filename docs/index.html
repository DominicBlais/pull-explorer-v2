    Pull Explorer v2

    Pull Explorer is a very simple demonstration library that analyzes GitHub pull requests and provides basic statistics about these PRs. It is not intended as a production system, but instead is simply a very small code style demonstration.

    Building

    To build Pull Explorer v2 from source, use npm run build after testing via npm test or npm test:cover. Both minified and uncompressed versions will be built. To build the API documentation, use npm run docs.
